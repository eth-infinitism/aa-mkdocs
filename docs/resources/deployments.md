# Official ERC-4337 Deployments

## Security

Specifying the incorrect `EntryPoint` address will result in a catastrophic loss of funds.
Double-check the address before using it across multiple trusted sources.
The source code of the `EntryPoint` contracts is available on [GitHub](https://github.com/eth-infinitism/account-abstraction).


## Deterministic Addresses

The `EntryPoint` contracts are always deployed using a `CREATE2` factory,
leading to it having the same address on all supported EVM-compatible chains.
